### 📝 10道布尔逻辑小测试

#### 🔹 第一部分：Python 阵营（第 1 - 7 题）

**1.** 在 Python 中，以下表达式的返回结果是什么？

Python



```
print(3 > 5)
```

**2.** 在 Python 中，以下表达式的返回结果是什么？

Python



```
print(0 or "Hello" or 9)
```

**3.** 在 Python 中，以下表达式的返回结果是什么？

Python



```
print(10 and [] and "Python")
```

**4.** 在 Python 中，`not` 任何值都会返回布尔值。那么下面这个会返回什么？

Python



```
print(not "苹果")
```

**5.** 综合大招：看看这个混合了“裁判（比较符号）”和“搬运工（逻辑符号）”的 Python 表达式：

Python



```
print(5 > 3 or 0 or "测试")
```

**6.** 进阶思考：如果全军覆没，`or` 会返回最后一个值。那么下面这个结果是什么？

Python



```
print(0 or "" or False)
```

**7.** 仔细观察：下面这个 Python 表达式返回的是什么类型（数字还是布尔值）？

Python



```
print(10 == 10 and 5 > 2)
```

#### 🔹 第二部分：C 语言阵营（第 8 - 10 题）

**8.** 在 C 语言中，以下表达式的输出结果（`result` 的值）是什么？

C



```
int result = 5 > 3;
```

**9.** 在 C 语言中，非 0 为真。那么下面这个表达式的输出结果（`result` 的值）是什么？

C



```
int result = 0 || 4 || 6;
```

**10.** 终极大挑战：在 C 语言中，`!` 代表取反（相当于 Python 的 `not`）。下面这个表达式的输出结果是什么？

C



```
int result = !(-5);
```

### 🛑 请先做题，完成后再向下滑动看答案和解析！









#### 1. 答案：`False`

- **解析：** `>` 是数学比较符号（铁面裁判），只负责判对错。3 大于 5 不成立，所以返回 `False`。

#### 2. 答案：`"Hello"`

- **解析：** `or` 是数值搬运工，找第一个“真”值。`0` 是假，继续往右；`"Hello"` 是非空字符串，为真！立刻短路停下，原物奉还，返回 `"Hello"`。

#### 3. 答案：`[]`

- **解析：** `and` 是数值搬运工，找第一个“假”值。`10` 是真，继续往右；`[]` 是空列表，在 Python 里代表假！立刻短路停下，原物奉还，返回 `[]`。

#### 4. 答案：`False`

- **解析：** `not` 会强制将右边的值转为布尔值并取反。`"苹果"` 是真，取反后就是 `False`。

#### 5. 答案：`True`

- **解析：** 1. 先算 `5 > 3`，裁判给出结果 **True**。

  \2. 表达式变成了 `True or 0 or "测试"`。

  \3. `or` 开始从左往右找第一个真值，第一眼就看到了 `True`（它本身就是真）。

  \4. 立刻短路停下，把这个 `True` 搬运出来。结果就是 `True`。

#### 6. 答案：`False`

- **解析：** `or` 找真值。`0` 是假，`""` 是假，最后遇到了 `False` 也是假。由于全军覆没，`or` 只能把最后一个踩到的值丢出来，也就是布尔值的 `False`。

#### 7. 答案：`True`（布尔值）

- **解析：** 1. `10 == 10` 结果是 `True`。

  \2. `5 > 2` 结果是 `True`。

  \3. 表达式变成 `True and True`。

  \4. `and` 找假值，没找到，于是返回最后一个真值，即第二个 `True`。所以它是一个布尔值。

#### 8. 答案：`1`

- **解析：** 欢迎来到 C 语言非黑即白的世界！5 大于 3 成立（真），C 语言固定返回 `1`。

#### 9. 答案：`1`

- **解析：** C 语言也有短路机制。从左到右，`0` 是假，继续；查到 `4` 时为真（非0即真）。由于是 `||` (or) 运算，此时整个表达式判定为真。C 语言很大气地收下 `4`，但极其吝啬地只吐出代表真的固定数字 `1`。

#### 10. 答案：`0`

- **解析：** 在 C 语言中，`-5` 是非零数，所以它代表**真**。对“真”进行 `!`（取反）操作，结果就是“假”。而 C 语言中“假”的唯一代言人就是 **0**。
